# Company Database SQL Project

## 📌 Overview
This project contains a complete SQL database design with multiple related tables.

## 🛠️ Features
- Employee management system
- Departments, Jobs, Projects
- Salary tracking
- Leaves and performance reviews
- Training and assets management

## 📂 Files
- schema.sql → Database structure
- data.sql → Insert data
- queries.sql → Practice queries

## 💡 Skills Used
- SQL (DDL, DML)
- Joins
- Subqueries
- Window Functions
- CTEs
- Constraints & Indexes

## 🚀 How to Run
1. Run schema.sql
2. Run data.sql
3. Run queries.sql
